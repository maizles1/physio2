# 🎯 מה צריך לעשות עכשיו - הסבר קצר ופשוט

## ✅ מה כבר נעשה (אני כבר עשיתי):

1. ✅ הקוד כבר מוכן - הוא יודע לטעון ביקורות מ-Google
2. ✅ הקובץ `.env.local` כבר נוצר
3. ✅ הכל מוכן לעבוד

---

## ⚠️ מה **אתה** צריך לעשות (2 דברים):

### 1️⃣ מצא את ה-Place ID (5 דקות)

**מה זה?** - מספר מזהה של העסק שלך ב-Google

**איך מוצאים?**
1. פתח Google Maps: https://www.google.com/maps
2. חפש: "יקינטון 3, אשדוד" (או שם הקליניקה)
3. לחץ על שם הקליניקה
4. לחץ "שתף" (Share)
5. לחץ "העתק קישור"
6. פתח מסמך והדבק את הקישור
7. חפש בקישור את המחרוזת שמתחילה ב-`ChIJ...`
   - זה ה-Place ID!
   - העתק רק את החלק הזה

**או:**
- הרץ: `npm run find-place-id` (אם יש לך כבר API Key)

---

### 2️⃣ צור API Key (10 דקות)

**מה זה?** - מפתח גישה ל-Google (כמו סיסמה)

**איך יוצרים?**
1. לך ל: https://console.cloud.google.com/
2. צור פרויקט חדש → תן שם → Create
3. חפש "Places API" → לחץ Enable
4. לך ל: "APIs & Services" → "Credentials"
5. לחץ "+ CREATE CREDENTIALS" → "API Key"
6. העתק את המפתח שנוצר (מתחיל ב-`AIza...`)
7. (מומלץ) לחץ על המפתח → "API restrictions" → בחר "Places API" → Save

---

## ✏️ הוספה לקובץ

**פתח את הקובץ:** `.env.local` בתיקיית הפרויקט

**מצא את השורות:**
```
NEXT_PUBLIC_GOOGLE_PLACE_ID=
GOOGLE_MAPS_API_KEY=
```

**הוסף את הערכים:**
```
NEXT_PUBLIC_GOOGLE_PLACE_ID=ChIJ... (החלף ב-Place ID שמצאת)
GOOGLE_MAPS_API_KEY=AIza... (החלף ב-API Key שיצרת)
```

**שמור את הקובץ**

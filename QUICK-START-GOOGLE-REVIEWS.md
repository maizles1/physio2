# ⚡ התחלה מהירה: הוספת ביקורות מ-Google

## 🎯 מה המטרה?

להוסיף את הביקורות מ-Google My Business שלך לאתר, כדי שיופיעו אוטומטית בדף העדויות.

---

## 📝 מה צריך? (2 דברים בלבד)

### 1️⃣ **Place ID** - המספר המזהה של העסק שלך ב-Google
### 2️⃣ **API Key** - מפתח גישה ל-Google

---

## 🔍 שלב 1: מציאת Place ID (5 דקות)

### הדרך הכי פשוטה:

1. **פתח Google Maps** → https://www.google.com/maps

2. **חפש את הקליניקה שלך** (למשל: "פיזיותרפיה אשדוד")

3. **לחץ על שם הקליניקה** כדי לפתוח אותה

4. **לחץ על כפתור "שתף" (Share)** ← בצד שמאל

5. **לחץ על "העתק קישור"** ← בחלון שנפתח

6. **פתח מסמך חדש** (TextEdit, Notes וכו') ו**הדבק את הקישור**

7. **חפש בקישור מחרוזת שמתחילה ב-`ChIJ`** ← זה ה-Place ID!

   📌 **דוגמה לקישור:**
   ```
   https://www.google.com/maps/place/.../data=!3m1!4b1!4m6!3m5!1sChIJ1234567890!8m2!3d...
                                                                     ^^^^^^^^^^^^^^^^
                                                                     זה ה-Place ID!
   ```

8. **העתק רק את החלק הזה** (מתחיל ב-ChIJ ומסתיים לפני הסימן הבא)

---

## 🔑 שלב 2: יצירת API Key (10 דקות)

### שלבים:

#### א. יצירת פרויקט:

1. **פתח:** https://console.cloud.google.com/
2. **התחבר** עם חשבון Google שלך
3. **לחץ על שם הפרויקט** למעלה (או "Select a project")
4. **לחץ "New Project"**
5. **תן שם:** "Physiotherapy Website" (או כל שם)
6. **לחץ "Create"**

#### ב. הפעלת Places API:

1. **בחיפוש למעלה**, חפש: **`Places API`**
2. **לחץ על "Places API"**
3. **לחץ על כפתור "ENABLE"** (כחול גדול)
   - ✅ זה בחינם עד 5000 בקשות בחודש!

#### ג. יצירת מפתח API:

1. **בצד שמאל**, לחץ: **"APIs & Services"** → **"Credentials"**
2. **לחץ על הכפתור הכחול:** **"+ CREATE CREDENTIALS"**
3. **בחר:** **"API Key"**
4. **תוצג הודעה עם המפתח** ← **העתק אותו עכשיו!**
   - נראה כך: `AIzaSyD1234567890abcdefghijklmnopqrstuvwxyz`
   - ⚠️ **שמור אותו במקום בטוח!**

#### ד. (מומלץ) הגבלת המפתח:

1. **לחץ על המפתח שיצרת** (בטבלה)
2. תחת **"API restrictions"**:
   - בחר **"Restrict key"**
   - בחר רק: **"Places API"**
3. **לחץ "SAVE"**

## ✏️ שלב 3: הוספה לקובץ

### פתח את הקובץ `.env.local`

1. **לך לתיקיית הפרויקט** `/Users/andrey/Documents/physio2`
2. **פתח את הקובץ:** `.env.local` (יכול להיות מוסתר - לחץ Cmd+Shift+. להציג קבצים מוסתרים)
3. **מצא את השורות:**
   ```
   NEXT_PUBLIC_GOOGLE_PLACE_ID=
   GOOGLE_MAPS_API_KEY=
   ```
4. **הוסף את הערכים אחרי הסימן `=`** (ללא רווחים לפני/אחרי)

   **לפני:**
   ```
   NEXT_PUBLIC_GOOGLE_PLACE_ID=
   GOOGLE_MAPS_API_KEY=
   ```

   **אחרי:**
   ```
   NEXT_PUBLIC_GOOGLE_PLACE_ID=ChIJ1234567890
   GOOGLE_MAPS_API_KEY=AIzaSyD1234567890abcdefghijklmnopqrstuvwxyz
   ```

5. **שמור את הקובץ** (Cmd+S)

## ✅ שלב 4: בדיקה

**פתח טרמינל והרץ:**
```bash
npm run check-reviews
```

### אם הכל תקין, תראה:
```
✅ קובץ .env.local נמצא
✅ NEXT_PUBLIC_GOOGLE_PLACE_ID: ChIJ...
✅ GOOGLE_MAPS_API_KEY: AIza...
✅ חיבור ל-Google Places API הצליח!
   שם העסק: פיזיותרפיה.פלוס
   דירוג ממוצע: 4.8 ⭐
   ביקורות זמינות: 5
```

### אם יש בעיה, תראה:
```
❌ שגיאת API: REQUEST_DENIED
```
← זה אומר שה-API Key לא תקין או שהפעלת את Places API

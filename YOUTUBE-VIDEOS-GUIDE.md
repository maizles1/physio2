# מדריך הוספת סרטוני YouTube לאתר

## 🎉 הדרך הקלה ביותר - משתני סביבה

עכשיו **הדרך הכי פשוטה** להוסיף סרטוני YouTube היא באמצעות משתני סביבה!

## איך זה עובד?

פשוט הוסף את קישורי ה-YouTube שלך לקובץ `.env.local` - אין צורך לגעת בקוד!

## הוראות שלב אחר שלב

### שלב 1: צור קובץ `.env.local`

צור קובץ חדש בשם `.env.local` בתיקיית השורש של הפרויקט (באותה תיקייה שבה נמצא `package.json`).

### שלב 2: הוסף את קישורי ה-YouTube שלך

פתח את הקובץ `.env.local` והוסף את השורות הבאות:

```env
# סרטון 1 - סרטון על הקליניקה
VIDEO_1_URL=https://www.youtube.com/watch?v=YOUR_VIDEO_ID_1

# סרטון 2 - על אנדריי מייזלס
VIDEO_2_URL=https://www.youtube.com/watch?v=YOUR_VIDEO_ID_2

# סרטון 3 - סיפור הצלחה
VIDEO_3_URL=https://www.youtube.com/watch?v=YOUR_VIDEO_ID_3
```

**החלף את `YOUR_VIDEO_ID_1`, `YOUR_VIDEO_ID_2`, `YOUR_VIDEO_ID_3` בקישורי ה-YouTube שלך!**

### שלב 3: כל סוג קישור עובד!

**תוכל להשתמש בכל אחד מהפורמטים הבאים:**

✅ `https://www.youtube.com/watch?v=VIDEO_ID`  
✅ `https://youtu.be/VIDEO_ID`  
✅ `https://www.youtube.com/embed/VIDEO_ID`

**האתר ימיר אוטומטית כל אחד מהם לפורמט הנכון!**

### שלב 4: שמור והפעל מחדש

1. שמור את הקובץ `.env.local`
2. הפעל מחדש את השרת (`npm run dev`)
3. רענן את הדפדפן

## דוגמה מלאה

```env
# סרטון 1 - סרטון על הקליניקה
VIDEO_1_URL=https://www.youtube.com/watch?v=dQw4w9WgXcQ

# סרטון 2 - על אנדריי מייזלס
VIDEO_2_URL=https://youtu.be/dQw4w9WgXcQ

# סרטון 3 - סיפור הצלחה
VIDEO_3_URL=https://www.youtube.com/embed/dQw4w9WgXcQ
```

**כל אחד מהם יעבוד!**

## יתרונות השיטה החדשה

1. **פשוט יותר** - אין צורך לגעת בקוד
2. **בטוח יותר** - הקובץ `.env.local` לא נשמר ב-git
3. **נוח יותר** - קל לעדכן ולשנות
4. **גמיש** - תומך בכל סוגי הקישורים

## אם לא הגדרת משתני סביבה

אם לא הגדרת משתני סביבה, האתר ישתמש ב-PLACEHOLDER והסרטונים לא יוצגו.  
זה בסדר - פשוט הוסף את הקישורים כשתהיה מוכן!

## איך למצוא את הקישור?

1. פתח את הסרטון ב-YouTube
2. לחץ על "שתף"
3. העתק את הקישור
4. הדבק בקובץ `.env.local`

## שאלות?

- ראה `ENV-SETUP.md` למדריך מפורט יותר
- ראה `.env.example` לדוגמה מוכנה להעתקה

## תכונות נוספות

1. **Modal להצגת סרטונים:**
   - הסרטונים נפתחים במודל יפה באתר עצמו
   - לא פותחים עוד דפים חדשים
   - סגירה עם Escape או לחיצה מחוץ לסרטון

2. **תמונות Thumbnail אוטומטיות:**
   - אם אתה מכניס קישור YouTube תקין, האתר יטען אוטומטית את תמונת התצוגה מ-YouTube
   - אין צורך להעלות תמונות ידנית!

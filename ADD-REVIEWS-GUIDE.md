# 📝 מדריך: איך להוסיף ביקורות ידנית לאתר

## 🎯 מה זה?

במקום טעינה אוטומטית, אתה מוסיף ביקורות ידנית ישירות לקובץ. זה פשוט ומהיר יותר!

---

## 📂 איפה מוסיפים?

**קובץ:** `config/manual-reviews.config.ts`

---

## ✏️ איך להוסיף ביקורת? (3 שלבים)

### שלב 1: מצא ביקורת ב-Google My Business

1. לך ל-Google My Business שלך
2. פתח את הביקורות
3. בחר ביקורת שתרצה להוסיף
4. העתק את הפרטים:
   - **שם המחבר**
   - **דירוג** (מספר הכוכבים: 1-5)
   - **תאריך** (למשל: "15 בינואר 2025")
   - **זמן יחסי** (למשל: "לפני שבוע", "לפני חודשיים")
   - **תוכן הביקורת** (הטקסט המלא)

### שלב 2: פתח את הקובץ

פתח את הקובץ: `config/manual-reviews.config.ts`

### שלב 3: הוסף ביקורת

הוסף ביקורת חדשה בתוך המערך `manualReviews`:

```typescript
{
  id: 'review-1', // מזהה ייחודי - שנה לכל ביקורת (review-1, review-2, וכו')
  authorName: 'שם המחבר', // שם המחבר מ-Google
  rating: 5, // מספר בין 1-5
  date: '15 בינואר 2025', // תאריך הביקורת
  relativeTimeDescription: 'לפני שבוע', // "לפני שבוע", "לפני חודש" וכו'
  text: 'תוכן הביקורת כאן...', // הטקסט המלא של הביקורת
},
```

## ✅ טיפים חשובים

1. **מספר ID ייחודי:** כל ביקורת צריכה `id` שונה
   - השתמש במספרים: `review-1`, `review-2`, `review-3` וכו'

2. **תאריכים:**
   - אפשר לכתוב תאריך מלא: `"15 בינואר 2025"`
   - או זמן יחסי: `"לפני שבוע"`, `"לפני חודשיים"`

3. **סדר הביקורות:**
   - הביקורות יוצגו לפי הסדר שהוספת
   - הראשונה שכתובה תופיע ראשונה

4. **תמונות פרופיל (אופציונלי):**
   - אם תרצה, תוכל להוסיף `profilePhotoUrl: 'https://...'`
   - אם לא תוסיף, יוצגו ראשי תיבות של השם

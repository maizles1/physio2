## 🚀 איך להעלות?

### דרך Finder (Mac)
1. פתח Finder
2. לחץ `Cmd+Shift+G`
3. הדבק: `/Users/andrey/Documents/physio2/public/images/services/tmj/`
4. העתק את התמונה ושנה את השם ל: `service-image.jpg`

### דרך Cursor/VS Code
1. פתח את התיקייה `public/images/services/tmj/` בפרויקט
2. גרור את התמונה לתיקייה
3. שנה את השם ל: `service-image.jpg`

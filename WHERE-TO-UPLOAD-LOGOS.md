## 🚀 איך להעלות?

### שיטה 1: דרך Finder (Mac)

1. פתח **Finder**
2. לחץ `Cmd+Shift+G` (או לך ל-Go > Go to Folder)
3. הדבק את הנתיב: `/Users/andrey/Documents/physio2/public/images/logo/`
4. העתק את הלוגו שלך לתיקייה
5. שנה את השם ל: `clinic-logo.png`

### שיטה 2: דרך VS Code / Cursor

1. פתח את התיקייה `public/images/logo/` בפרויקט
2. לחץ ימני > New File
3. או פשוט גרור את קובץ הלוגו לתיקייה
4. ודא שהשם נכון

## ✅ איך יודעים שהלוגו נטען?

1. **שמור את הקובץ** בתיקייה הנכונה
2. **רענן את הדפדפן** (Cmd+R או Ctrl+R)
3. **בדוק:**
   - הלוגו שלך מופיע ב-Header (כותרת עליונה) במקום ה-placeholder
   - לוגואי הקופות מופיעים ברובריקת "קופות חולים וביטוחים"

## 📝 הערות חשובות

- **שמות קבצים:** חייבים להיות בדיוק כמו שמופיע בטבלה (case-sensitive!)
- **פורמטים:** PNG, SVG, או JPG - כולם עובדים
- **גודל:** הלוגו שלך יכול להיות כל גודל - האתר יעשה אופטימיזציה
- **Fallback:** אם הלוגו לא קיים, האתר יציג placeholder

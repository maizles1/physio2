# 📝 איך להוסיף ביקורות ידנית לאתר

## 🎯 מה זה?

במקום טעינה אוטומטית מ-Google, אתה מוסיף ביקורות ידנית בקובץ פשוט.

---

## 📂 איפה מוסיפים?

**קובץ:** `config/manual-reviews.config.ts`

---

## ✏️ איך להוסיף ביקורת?

### שלב 1: מצא ביקורת ב-Google My Business

1. לך ל-Google My Business שלך
2. פתח את הביקורות
3. העתק את הפרטים:
   - **שם המחבר**
   - **דירוג** (1-5 כוכבים)
   - **תאריך** (למשל: "15 בינואר 2025" או "לפני שבוע")
   - **תוכן הביקורת**

### שלב 2: הוסף לקובץ

פתח את הקובץ: `config/manual-reviews.config.ts`

הוסף ביקורת חדשה כך:

```typescript
{
  id: 'review-1', // מזהה ייחודי - שנה לכל ביקורת
  authorName: 'שם המחבר', // שם המחבר מ-Google
  rating: 5, // מספר בין 1-5
  date: '15 בינואר 2025', // תאריך הביקורת
  relativeTimeDescription: 'לפני שבוע', // "לפני שבוע", "לפני חודש" וכו'
  text: 'תוכן הביקורת כאן...', // הטקסט המלא של הביקורת
},
```

---

## ✅ טיפים

1. **מספר ID ייחודי:** כל ביקורת צריכה `id` שונה (review-1, review-2, וכו')
2. **תאריכים:** אפשר לכתוב תאריך מלא או "לפני שבוע", "לפני חודשיים"
3. **סדר:** הביקורות יוצגו לפי הסדר שהוספת (הראשונה תופיע ראשונה)

# הוראות הפעלה - אתר פיזיותרפיה.פלוס

## איך להפעיל את האתר?

### שלב 1: פתח את הטרמינל

### שלב 2: נווט לתיקיית הפרויקט
```bash
cd /Users/andrey/Documents/physio2
```

### שלב 3: הפעל את השרת
```bash
npm run dev
```

### שלב 4: פתח את הדפדפן
פתח את הדפדפן שלך וגש לכתובת:

**http://localhost:3000**

או:

**http://127.0.0.1:3000**

## מיקום הפרויקט
הפרויקט נמצא בתיקייה:
```
/Users/andrey/Documents/physio2
```

## הערות חשובות
- השרת צריך להיות רץ כדי שהאתר יעבוד
- אחרי הפעלת `npm run dev`, תמתין כמה שניות עד שתראה הודעה "Ready"
- אם הפורט 3000 תפוס, השרת יעבור אוטומטית לפורט 3001

## עצירת השרת
לעצור את השרת, לחץ `Ctrl + C` בטרמינל
